# Algorytmy i Struktury Danych 2

Rozwiązania zadań z przedmiotu **Algorytmy i Struktury Danych 2**, realizowanego w ramach czwartego semestru studiów na kierunku Informatyka i Systemy Informacyjne na wydziale Matematyki i Nauk Informacyjnych.

## Zrealizowane laboratoria:

* **Lab01** - Dynamic programming (programowanie dynamiczne)
* **Lab03** - Graphs algorithms (algorytmy grafowe)
* **Lab04** - Shortest path finding (znajdowanie najkrótszej ścieżki)
